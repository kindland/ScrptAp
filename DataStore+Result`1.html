Version Property

Version of the value for the key used for this operation.

Syntax

[get: Sansar.Script.Interface]
public int Version { get; }

Value

An integer version number for the data in the store.

Remarks

Each write to a given key will update this value, even if the same value is written. To ensure that one script is not clobbering writes by another script the Sansar.Simulation.DataStore.Options.Version can verify that the data has not changed.
