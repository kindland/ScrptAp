CharacterComponent Class

The CharacterComponent handles character specific operations like playing CharacterAnimations.

Syntax

[Sansar.Script.Interface]
public class CharacterComponent : Sansar.Script.InstanceInterface

Remarks

Allows for doing operations specific to characters using the Sansar skeleton types

PlayAnimation Method

Plays an animation in server time synchronously for this CharacterComponent. If you want multiple agent's to play an animation together, use this function

Syntax

Parameters

animation
The animation to play on the agent.
playbackMode
How to play the animation. Only PlayOnce and Loop supported.
skeletonSubset
Which subset of the skeleton to play the animation on.

Returns

The return value is an internal event id.

Remarks
